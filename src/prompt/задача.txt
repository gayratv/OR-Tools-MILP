Для символьной работы: можно описывать уравнения как формулы, потом автоматически превращать их в матрицы.

Удобно для «чисто математических» постановок.

================
python
мне нужно составить систему линейных уравнений для последующего решения бесплатным солвером MILP
моя задача не коммерческая

общее описание: составить школьное расписание

Исходные данные будут загружаться из базы данных
Этап загрузки данных из базы данных пропустим

1. Есть предметы (загружаемые из базы данных), например
 математика, информатика, английский,труд

2. есть учителя,  учитель может преподавать один или несколько предметов, например Иванов может преподавать алгебру и геометрию

3. есть года обучения от 5 до 11

4. есть классы, например 5а, 5б

5. есть учебный план в котором указано что для 5 класса должно быть X часов (целое число) уроков в неделю по предмету П

6. за каждым классом по каждому предмету закреплен конкретный учитель

7. учеба проводится по 5 дней в неделю, не более 7 уроков в день

8. каждый учитель может вести не более 35 уроков в неделю

9. дополнительные ограничения
каждый учитель может выбрать дни , в которые он не ведет уроки (например Петров не ведет уроки в понедельник)

10. не больше 1 урока одного предмета в день на класс

Задача:
напиши код на python, который будет генерировать систему линейных уравнений

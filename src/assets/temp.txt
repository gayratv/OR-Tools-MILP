
teacher_forbidden_slots — словарь запретов по слотам для конкретных преподавателей.
max_repeats_per_day — максимум повторений предмета в день (bool / int / dict).
min_days_per_subject — минимум разных дней в неделю для конкретного (class, subject).
must_sync_split_subjects — множество split-предметов, которые должны идти синхронно в подгруппах.
max_consecutive_lessons_for_class — максимум подряд идущих уроков для класса.
max_consecutive_lessons_for_teacher — максимум подряд идущих уроков для учителя.


max_repeats_per_day

Задаёт, сколько раз предмет может встречаться у одного класса в течение дня.

Вариант (bool):

max_repeats_per_day = True   # любой предмет не чаще 1 раза в день


Вариант (int):

max_repeats_per_day = 2      # любой предмет не чаще 2 раз в день


Вариант (dict):

max_repeats_per_day = {
    "math": 1,      # математика максимум 1 урок в день
    "eng": 2,       # английский можно до 2 раз в день
}

min_days_per_subject

Словарь: ключ — (класс, предмет), значение — минимум разных дней в неделю, когда должен быть этот предмет.

min_days_per_subject = {
    ("5A", "math"): 2,   # в 5А математика должна быть хотя бы в 2 разных днях
    ("5B", "eng


must_sync_split_subjects

Множество split-предметов, у которых подгруппы должны идти синхронно (одновременно в слоте).

must_sync_split_subjects = {"eng"}   # английский: обе подгруппы в один и тот же слот


max_consecutive_lessons_for_class

Максимум подряд идущих уроков для класса. Может быть числом или словарём.

Вариант (int):

max_consecutive_lessons_for_class = 5   # у любого класса не более 5 уроков подряд


Вариант (dict):

max_consecutive_lessons_for_class = {
    "5A": 4,   # для 5А максимум 4 подряд
    "5B": 6,   # для 5Б можно до 6 подряд
}

max_consecutive_lessons_for_teacher

То же самое, но для учителей.

Вариант (int):

max_consecutive_lessons_for_teacher = 4   # у любого учителя не более 4 подряд


Вариант (dict):

max_consecutive_lessons_for_teacher = {
    "Ivanov": 3,    # Иванов не больше 3 подряд
    "Petrov": 5,    # Петров до 5 подряд
}

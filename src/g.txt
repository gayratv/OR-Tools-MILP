Вот список полей из `class InputData`, которые **не генерируются** в `generate_static_data_file.py` (то есть их нет в `create_timetable_data()`):


3. **grade\_max\_lessons\_per\_day**
   Ограничение максимального числа уроков в день для каждой параллели. Пример: `{2: 4, 3: 5, 4: 5}`.

4. **subjects\_not\_last\_lesson**
   Предметы, которые запрещено ставить последними уроками для указанной параллели. Пример: `{5: {"math"}, 7: {"math", "physics"}}`.

5. **elementary\_english\_periods**
   Разрешённые номера уроков для английского в начальной школе. Пример: `{2, 3, 4}`.

6. **grade\_subject\_max\_consecutive\_days**
   Ограничения по максимальному числу подряд идущих дней, когда у параллели может быть один и тот же предмет. Пример: `{3: {"PE": 2}}`.

7. **must\_sync\_split\_subjects**
   Набор сплит-предметов, которые должны вестись синхронно у всех подгрупп. Например: `{"eng", "cs"}` означает, что английский и информатика должны идти одновременно.
